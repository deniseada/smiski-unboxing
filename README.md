# Smiski Digital Unboxing Experience

## Overview

This project is an **interactive digital Smiski blind box unboxing experience**. Blind box toys have become increasingly popular because buyers do not know which figure they will receive until the box is opened, creating excitement, suspense, and surprise.

This website recreates that same emotional experience in a **digital format**. Users will be able to choose a virtual box, open it, and reveal a randomized Smiski character. Through animations, sound effects, and interactive elements, the goal is to simulate the same dopamine-driven excitement that comes from opening a real blind box.

The project focuses on combining **UI design, animation, and randomized logic** to create a playful and engaging web experience.

## What I Am Building

The website will include the following key features:

### Landing Page
A visually engaging landing page inspired by the **Smiski brand aesthetic**, introducing users to the blind box experience.

### Interactive Blind Box Selection
Users will be able to **click and choose a virtual blind box** to open.

### Animated Unboxing
The box will animate when opened, possibly including:

- Shaking motion  
- Opening animation  
- Suspenseful reveal  

### Randomized Smiski Reveal
A randomization system will determine which Smiski figure appears based on **different rarity tiers**.

### Collection Tracker
A system that tracks which figures users have revealed and how many times duplicates appear.

## Project Goal

The primary goal is to **replicate the emotional experience of opening a real blind box**, translating the curiosity, suspense, and excitement into a digital interactive format.
